PREFIX EXPRESSION

The prefix expression is an expression in which the operator is written before the operands.

For example,
* +pq +rs

Syntax of prefix notation is given below:
< operator > < operand > < operand >

POSTFIX EXPRESSION

The postfix expression is an expression in which the operator is written after the operands.

For example,
pq+ rs+ *

Syntax of postfix notation is given below:
< operand > < operand > < operator >

ALGORITHM FOR PREFIX TO POSTFIX EXPRESSION

Let, Q be the arithematic expression written in prefix notation. P is the postfix notation.

Step 1: Start
Step 2: Reverse the contents of Q
Step 3: Scan Q from left to right and repeat steps 4 to 5 until Q is completely scanned.
Step 4: If the symbol is an operator,
A. Pop the top element from the stack and store it in operator 1.
B. Pop the next top element from the stack and store it in operator 2.
C. Copy operator 1 into P and add operator 2 into P.
D. Join symbol into P.
E. Push P into stack.
Step 5:If it is not an operator
A. Push symbol into stack.
Step 6:END.
